# Cultivation Game

An incremental / idle cultivation game inspired by xianxia themes.

## Running

Open `index.html` in your browser. No build tools or frameworks are required.

## Testing

Run a simple smoke test to ensure the page serves correctly:

```bash
bash scripts/smoke-test.sh
```

## Features

- Stylized 3D-inspired cultivation scene built with HTML/CSS only
- Idle qi generation with active actions to meditate, study, and train
- Stage breakthroughs with resource requirements and setback chance
- Exploration events that reward different attributes
